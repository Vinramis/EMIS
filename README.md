# Инструкция EMIS Automator

***! ! ! Перед началом или при возникновении трудностей прошу прочитать ЧаВо (Частые вопросы) в конце.***

Также прошу в любом случае поделиться опытом использования — это поможет сделать программу лучше

## Установка и настройка

1. Скачайте архив (последнюю версию можно найти в телеграм канале по тегу **@emis_automator** или по ссылке **t.me/emis_automator**)
2. Распакуйте архив в удобную для вас папку или на рабочий стол
3. В родительской папке (emis automator) найдите папки **КЛ** и **ДЗ**
4. Удалите из этих папок файлы-примеры и перенесите соответствующие файлы

## Использование

1. Запустите файл **СТАРТ.cmd**
2. Следуйте инструкциям в окне консоли
3. Полсе того, как в окне консоли появится сообщение "**Автоматизация завершена!**", заполните **4 поля** в начале страницы EMIS (Sarlavha, Fan nomi, Oquv yili, Semestr)
4. **Проверьте, все ли файлы были загружены и загружены верно** Автоматором
5. Нажмите кнопку сохранения (Saqlash)

### ЧаВо (Частые Вопросы)

- Как мне переместить сразу все файлы из папки А в папку Б?

Ответ: В **папке А** один раз нажмите на любой файл, нажмите **Ctrl + A**, затем **Ctrl + C**, затем перейдите в **папку Б** и нажмите **Ctrl + V**

- Что значит "нажмите **Клавиша1 + Клавиша2**"?

Ответ: Это значит, что нужно *зажав* **Клавишу1** *нажать* на **Клавишу2**

- Как мне удалить все файлы из папки?

Ответ: В папке один раз нажмите на любой файл, затем нажмите **Ctrl + A**, затем нажмите **Delete**
