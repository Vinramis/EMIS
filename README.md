## Как делать файлы (Для тех, кто готовит задания)

1. Размер одиничного файла не должен превышать 10 мегабайт (Требование EMIS)
2. КТП должен быть в формате таблицы Excel. Темы начинаться с клетки B6
3. Файлы классных заданий должным быть отделены от файлов домашних заданий.
4. Название файла должно содержать порядковый номер его темы. Если чисел в названии несколько, то порядковый номер должен быть перед всеми остальными числами (например, если порядковый номер 23, файл может называться "23 Промежуточный контроль 2" или "кл_23_пк_2", но не "Промежуточный контроль 2 дз 23")

## ЧаВо (Частые Вопросы)

- Как мне переместить сразу все файлы из папки А в папку Б?

Ответ: В **папке А** один раз нажмите на любой файл, нажмите **Ctrl + A**, затем **Ctrl + C**, затем перейдите в **папку Б** и нажмите **Ctrl + V**

- Что значит "нажмите **Клавишу1 + Клавишу2**"?

Ответ: Это значит, что нужно *зажав* **Клавишу1** *нажать* на **Клавишу2**

- Что делать, если я случайно что-то удалил или вставил не туда?

Ответ: Просто нажмите **Ctrl + Z**

- Как мне удалить все файлы из папки?

Ответ: В папке один раз нажмите на любой файл, затем нажмите **Ctrl + A**, затем нажмите **Delete**
