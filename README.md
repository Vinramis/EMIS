## Настройка

1. Найдите родительскую папку (Automator), удалите оттуда имеющийся КТП и вставьте туда нужный КТП
2. Проверьте, какой файл старта в родительской папке. Если "**СТАРТ**", то назовите ваш КТП — "**КТП**". Если же ваш файл старта "**СТАРТ.cmd**" — назовите "**КТП.xlsx**"
3. В родительской папке (Automator) найдите папки **КЛ** и **ДЗ**
4. Удалите из этих папок файлы-примеры и переместите туда необходимые вам файлы

## Как делать файлы (Для тех, кто готовит задания)

##### Обязательно

1. Размер одиничного файла не должен превышать 10 мегабайт (Это — требование EMIS)
2. Файлы классных заданий должным быть отделены от файлов домашних заданий. ***Это — самое важное*** для работы программы

##### Желательно

1. Название файла должно начинаться на порядковый номер его темы
2. После порядкового номера должен быть пробел

## ЧаВо (Частые Вопросы)

- Как мне переместить сразу все файлы из папки А в папку Б?

Ответ: В **папке А** один раз нажмите на любой файл, нажмите **Ctrl + A**, затем **Ctrl + C**, затем перейдите в **папку Б** и нажмите **Ctrl + V**

- Что значит "нажмите **Клавишу1 + Клавишу2**"?

Ответ: Это значит, что нужно *зажав* **Клавишу1** *нажать* на **Клавишу2**

- Что делать, если я случайно что-то удалил или вставил не туда?

Ответ: Просто нажмите **Ctrl + Z**

- Как мне удалить все файлы из папки?

Ответ: В папке один раз нажмите на любой файл, затем нажмите **Ctrl + A**, затем нажмите **Delete**
